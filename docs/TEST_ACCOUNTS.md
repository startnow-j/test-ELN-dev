## 测试场景

### 场景1：超级管理员功能测试
1. 使用 `superadmin@example.com` 登录
2. 访问用户管理 → 创建新用户、修改用户角色
3. 访问文件管理 → 查看孤立文件、清理孤立文件
4. 测试项目删除功能

### 场景2：管理员功能测试
1. 使用 `admin@example.com` 登录
2. 访问用户管理 → 创建普通用户（不能创建 SUPER_ADMIN）
3. 访问文件管理 → 查看存储统计、下载文件（不能清理孤立文件）

### 场景3：项目负责人角色测试
1. 使用 `lead@example.com` 登录
2. 创建项目 → 自动成为 PROJECT_LEAD
3. 添加成员 → 邀请研究员参与项目
4. 审核实验 → 审核项目内其他成员的实验

### 场景4：研究员角色测试
1. 使用 `researcher@example.com` 登录
2. 创建暂存实验 → 上传附件
3. 提交审核 → 选择审核人
4. 查看项目内容（需要先被添加到项目）
